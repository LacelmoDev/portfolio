# 🚀 LYV1 - Personal Portfolio

Este é o repositório do meu site portfólio pessoal, desenvolvido para centralizar meus projetos, habilidades e trajetória como desenvolvedor Full-Stack e entusiasta de cibersegurança.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído "à mão" para consolidar conhecimentos de front-end e design responsivo:
*   **HTML5**: Estrutura semântica e acessível.
*   **CSS3**: Estilização moderna com variáveis, Flexbox, Grid e efeitos de *Glassmorphism*.
*   **JavaScript**: Lógica para interatividade e integração com API de mensagens.
*   **FontAwesome**: Ícones para identidade visual técnica.

## 📱 Funcionalidades

*   **Design Responsivo**: Adaptado para ecrãs de telemóveis e computadores.
*   **Seção Sobre Mim**: Foco em liderança de equipas e atuação como estagiário.
*   **Showcase de Projetos**: Destaque para o Sistema Tarimba, PMS Hoteleiro e o utilitário LIVZIP.
*   **Contacto Direto**: Botão dinâmico integrado ao WhatsApp.
*   **Download de CV**: Acesso rápido ao currículo profissional.
